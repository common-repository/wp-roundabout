== Description ==

This plugin creates a round about slider based on jQuery Roundabout plugin.
You can either use the function wproundabout($cat, $noOfpost) in your theme 
or use the short [wproundabout category=1 number=4 ] to display the roundabout slider

== Installation ==

1. Upload Folder 'wp-roundabout' to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You are Ready :)
